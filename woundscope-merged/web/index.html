How WoundScope works

What this tool does

It reads every patient's records — insurance, diagnoses, and messy nurse notes — and decides who can be billed to Medicare Part B for wound care, with the reasoning and the source text behind each call.

The three decisions

Auto-Accept All rules clearly met — safe to send to billing.
Flag for Review Eligible, but something's missing or ambiguous — a human checks it.
Reject Not billable — no active wound, wrong insurance, or unreliable data.

How to read a patient

The evaluation log shows each billing rule as a pass (✓) or fail (✕): active wound → active Part B → wound reliably extracted → complete measurements (L/W/D) → drainage → confidence. Measurements turn red when missing. Wounds detected lists each wound when a note documents more than one (each bills separately). Evidence shows the exact source sentence; Provenance shows which record each value came from.

Privacy

Patient names are masked by default (minimum-necessary). Use the 🔒 toggle to reveal. The demo uses synthetic patients, so no real PHI is involved.
